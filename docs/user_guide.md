# FraudDetect 2.0 - Руководство пользователя

## Описание системы
FraudDetect 2.0 - это интеллектуальная система обнаружения финансового мошенничества, использующая машинное обучение для анализа транзакций и выявления подозрительной активности.

## Основные возможности
- Анализ транзакционных данных на предмет мошенничества
- Использование нескольких моделей машинного обучения
- Визуализация результатов анализа
- Экспорт отчетов в различных форматах
- Самообучающаяся система для улучшения точности

## Требования к данным
Файлы должны содержать следующие колонки:
- `step` - временной шаг транзакции
- `type` - тип транзакции (PAYMENT, TRANSFER, CASH_OUT, DEBIT, CASH_IN)
- `amount` - сумма транзакции
- `nameOrig` - идентификатор отправителя
- `oldbalanceOrg` - баланс отправителя до транзакции
- `newbalanceOrig` - баланс отправителя после транзакции
- `nameDest` - идентификатор получателя
- `oldbalanceDest` - баланс получателя до транзакции
- `newbalanceDest` - баланс получателя после транзакции

## Как использовать систему

### 1. Загрузка данных
1. Перетащите CSV или Excel файлы в область загрузки или нажмите для выбора файлов
2. Можно загрузить до 5 файлов одновременно
3. Система проверит формат данных и отобразит информацию о файлах

### 2. Настройка параметров анализа
В разделе "Настройки" слева можно настроить:
- **Модели ML**: выберите одну или несколько моделей для анализа:
  - Isolation Forest - быстрая модель для общего анализа
  - AutoEncoder - нейросеть для глубокого анализа
  - LSTM - анализ последовательностей для выявления паттернов
- **Параметры анализа**: установите ожидаемый уровень мошенничества (1-20%)
- **Тестовые данные**: выберите готовые наборы данных для тестирования

### 3. Подтверждение файлов
1. Выберите тип анализа:
   - Отдельно для каждого файла
   - Объединить все файлы
2. Нажмите "Подтвердить файлы и продолжить"

### 4. Анализ данных
Система автоматически проведет анализ и отобразит результаты:
- Ключевые показатели (всего транзакций, подозрительных, чистых)
- Распределение Fraud Score
- Вклад каждой модели в общий результат
- Подозрительные транзакции
- Анализ по типам транзакций
- Интеллектуальные рекомендации

### 5. Просмотр деталей
- Нажмите "Просмотр анализа" для каждого файла, чтобы увидеть подробные результаты
- Используйте вкладки для навигации по различным аспектам анализа

### 6. Экспорт результатов
В конце анализа можно экспортировать результаты в одном из форматов:
- HTML отчет
- CSV данные
- Excel данные
- JSON данные

## Интерпретация результатов

### Fraud Score
Чем выше значение Fraud Score, тем выше вероятность мошенничества. Система автоматически помечает транзакции с высоким Fraud Score как подозрительные.

### Модели анализа
- **Isolation Forest**: Быстрая модель, эффективная для общего обнаружения аномалий
- **AutoEncoder**: Нейросеть, которая учится восстанавливать нормальные транзакции и выявляет отклонения
- **LSTM**: Анализирует последовательности транзакций для выявления сложных паттернов

## Рекомендации по использованию
1. Для быстрого анализа начните с модели Isolation Forest
2. Для глубокого анализа используйте комбинацию всех моделей
3. Установите реалистичный уровень ожидаемого мошенничества (обычно 1-5%)
4. Внимательно изучите интеллектуальные рекомендации системы
5. Проверьте подозрительные транзакции вручную для подтверждения

## Устранение неполадок
- Если файл не загружается, проверьте формат данных
- Если анализ занимает слишком много времени, используйте только Isolation Forest
- Если нет подозрительных транзакций, возможно, уровень ожидаемого мошенничества установлен слишком низко
- При ошибках перезапустите приложение с помощью кнопки "Начать заново"
